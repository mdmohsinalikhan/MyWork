It is an Implementation of Groestl-256

File Descriptions:
----------------------

groestl256_driver.c 		Contains the main function and call the groestl implementation by sending message. It gets the calculated hash value 					of the sent message by a pointer to the hash value. It checks the hash value and prints PASS/FAIL

groestl256.c			Contains the implementation. It calulates the hash value of the message which was sent to it.

groestl256.h			Contains the headers
